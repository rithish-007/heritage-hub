# 🤝 Contributing to Heritage Hub

We love contributions! Here’s how you can help:

## 📝 Ways to Contribute
- Share local stories
- Help translate content
- Improve app features
- Fix bugs and issues
- Write documentation

## ⚙️ Development Setup
1. Fork the repo
2. Clone your fork
3. Create a branch (`git checkout -b feature-name`)
4. Make your changes
5. Commit with a clear message
6. Push and open a merge request

## 🧪 Code Style
- Use Python 3.9+
- Follow PEP8 guidelines
- Run `ruff` linter before commits

## 🐛 Reporting Bugs
Open an issue with:
- Description
- Steps to reproduce
- Expected vs actual behavior
